# KaloriN AI Testing Repository

QA testing repository for the KaloriN AI web application.

## Project Quick View

KaloriN AI is a nutrition tracking web app with:
- AI-powered food scanning
- Meal tracking
- Nutrition insights
- Personalized food recommendations
- BMI and calorie target calculation

This repository contains:
- Manual testing documentation
- Test scenarios
- Test cases
- Bug reports
- API testing
- Future automation testing setup

---

# Testing Scope

The testing process focuses on the main user flows and critical features of the application.

## Main Features Covered
- Authentication
- Protected Routes
- User Onboarding
- Food Search
- AI Food Scanner
- Meal Tracking
- Recommendations
- Weekly Insights
- Profile Management
